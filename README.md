# DSA Practice

This repository contains my daily Data Structures and Algorithms (DSA) practice in C++.

## Structure
Problems are organized by problem-solving patterns such as:
- Maths
- Two Pointers  
- Sliding Window  
- Binary Search  
- Dynamic Programming  
- hashing
- recursion and backtracking 
- miscellaneous
- stack
- queue
- tree
- graph
This helps improve pattern recognition and problem-solving skills.
